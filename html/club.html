Club Analysis

Background

Our club analysis focuses on pace, shooting, dribbling, passing, defending and physic. This is a basic performance statistic for all player positions, we decided to use a radar plot as a visualization type because it can help team managers access each component for each of the position between the 2 comparing teams. The user can zoom in and out from the radar plot by dragging the radar plot scale. The user can also rotate it so that they can get a clearer view of each composition measure. We decided to select Arsenal as our focus team for the Premier League club analysis

Findings & Results

We can see that the number of radar plot between each team comparison is not the same. This is because different teams have different formations and strategy; however, we were able to take a deep dive into the player composition of each similar position between the teams. These radar plot can be use in club manager's analysis and decision making to improve the overall club composition.
